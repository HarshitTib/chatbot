Inorder to run the file successfully, the steps need to be followed

1. Move the file "message.php" inside the xampp/htdocs folder which is located mainly in C:\xampp\htdocs

2. Create a database in Xampp and import "chatbot.sql" file in it.

3. To run the App.js, enter the command "npm start" in the terminal.

4. A chatbot will be displayed which can respond to basic queires of an user

Example of such queries are:
i. Take me to the Login page.
ii. Show me the blog.
iii. What are the courses offered?
iv. To know about the teams.
v. Deadline for the IIT-JAM exam.
vi. Deadline for the IAT exam.
vii. Study Materials.
